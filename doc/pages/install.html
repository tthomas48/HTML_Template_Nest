Installing

PEAR

Nest can be installed from the pearhub PEAR repository.

pear channel-discover tthomas48.github.com/pear
pear install tthomas48/HTML_Template_Nest

Composer

To add HTML_Template_Nest to Composer simply add the following line to your composer.json:

"buyplaytix/HTML_Template_Nest": "dev-master"

Github

You can find active development of the project on github.

https://github.com/tthomas48/HTML_Template_Nest/
